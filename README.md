# Swiggy Sales Data Analysis 

## Project Overview
This project is an interactive Swiggy Sales Analysis Dashboard developed using Power BI. It analyzes sales performance, customer ratings, order trends, food category preferences, and regional sales to provide business insights and support data-driven decision-making.

---

## Objectives.
- Analyze overall sales performance.
- Track monthly, weekly, and daily sales trends.
- Identify top-performing states.
- Analyze food category distribution.
- Monitor customer ratings and order value.
- Support business decisions using interactive dashboards.

---

## Tools Used
- Power BI
- Microsoft Excel
- Data Cleaning
- Data Visualization
- Pivot Table
- Pivot Chart

---

## Dashboard Features
- Total Sales
- Average Rating
- Total Orders
- Rating Count
- Average Order Value
- Monthly Sales Trend
- Daily Sales Trend
- Weekly Sales Trend
- Sales by Food Type
- Sales by State (Map)
- Top 5 Performing States
- Quarter-wise Sales Analysis
- Interactive Filters (Month, Restaurant, Category)

---

## Key Insights
- Total Sales: **₹53.01M**
- Average Customer Rating: **4.34**
- Average Order Value: **₹268.51**
- Non-Veg food category generated the highest sales.
- Maharashtra recorded the highest sales among all states.
- Sales remained relatively consistent across the week with slight fluctuations.
- Monthly sales showed strong performance in January and August.
